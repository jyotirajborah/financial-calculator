# Real Estate Empire: Survival Mode - Features

## Game Overview
A realistic, challenging real estate simulation game that replaces the old easy Monopoly mechanics.

## Key Features Implemented

### 1. Realistic Property System
- 8 properties ranging from $150 to $1,800
- Each property has:
  - **Rent**: Monthly income when occupied
  - **Maintenance**: Monthly costs
  - **Risk Level**: Low, Medium, or High (affects degradation)
  - **Condition**: 0-100% (degrades over time, affects value)

### 2. Dynamic Market Conditions
- **Boom** (1.3x property values, 5% vacancy rate)
- **Normal** (1.0x property values, 10% vacancy rate)
- **Recession** (0.8x property values, 20% vacancy rate)
- **Crash** (0.5x property values, 35% vacancy rate)

Market transitions happen randomly each month.

### 3. Financial Challenges
- Start with only $1,000 cash
- Can take loans up to $2,000 debt limit
- 8% annual interest (0.67% monthly) compounds on debt
- Properties degrade 1-3% per month based on risk level
- Vacancy risk increases during economic downturns

### 4. Random Events
- Emergency repairs ($200 cost)
- Property damage (20% condition loss)
- Bonus rent payments ($50)
- Events occur with 5-10% probability each month

### 5. Game Over Conditions
- Debt exceeds $5,000
- Bankrupt with no assets

### 6. Property Management
- **Buy**: Purchase properties (can use loans)
- **Repair**: Restore condition for $100 (adds 30% condition)
- **Sell**: Liquidate at 80% of current market value

### 7. Game Interface
- Real-time stats: Cash, Debt, Net Worth, Properties, Month, Market
- Available properties grid with current prices
- Your portfolio with condition and vacancy status
- Game log showing all events and transactions
- Next Month and New Game controls

## How to Play
1. Open the "Monopoly Game" tab
2. Buy properties strategically based on market conditions
3. Click "Next Month" to advance time
4. Manage your portfolio: repair properties, handle vacancies
5. Survive as long as possible without going bankrupt!

## Strategy Tips
- Buy during crashes when prices are low
- Maintain property condition to preserve value
- Watch your debt-to-cash ratio
- Diversify with different risk levels
- Sell properties before market crashes if needed
